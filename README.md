# React Landing Page

A responsive, modern landing page built with **React** for showcasing future updates and personal projects. Features a dark purple gradient theme, smooth animations, and a fully responsive design. Perfect for beginners learning React and front-end development.

---

## Key Features

- Built with **React** and CSS/JS animations  
- Responsive design for mobile and desktop  
- Dark purple gradient theme with smooth transitions  
- Lightweight and fast, ideal for a personal landing page  
- Easy to customize for personal branding  

---

## Tech Stack

- **React**  
- **HTML5 & CSS3**  
- **JavaScript**
